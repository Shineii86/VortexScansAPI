Features

Built for developers

Everything you need to build a manga reader website, mobile app, or Discord bot. No API keys, no rate limits, no hassle.

Full-text Search
Search by title with paginated results. Find manga, manhwa, and manhua by name instantly.
Chapter Images
Get direct image URLs for each chapter page from Vortex Scans CDN. No scraping needed.
Rich Metadata
Genres, ratings, types, statuses, chapter lists, and more. All the data you need in one API.
Advanced Filters
Filter by type (manhwa/manga/manhua), status (ongoing/hiatus/completed), and genre.
Home Data
Latest, hot, and top rated manga collections in a single endpoint. Perfect for landing pages.
Fast & Cached
In-memory cache with 5-minute TTL and max 100 entries. Snappy responses with reduced upstream requests.
